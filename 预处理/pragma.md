pragma
======
该单词本意是杂注的意思。它是一个比较复杂的预处理命令。  
它的基本格式为**#pragma para**。其中para为该命令的参数，下面我一一介绍各种参数
##message("")
在编译信息的输出窗口输出相应的信息。请注意并不是标准输入输出。
这个message也需要有一个字符串类型的参数。如：  
```c
#pragma message("I'm a coder!")
```
该宏定义可配合ifdef一起使用，则编译期来检查是否存在某个宏
##once
**#pragma once**。可以保证头文件只被编译一次，避免重复包含导致的冲突。
>另外一个经典地防止头文件重复包含的做法是：ifndef...define...endif方式。如：
```c
#ifndef _HELLO_H_
#define _HELLO_H_
...
#endif
```
上述做法中的_HELLO_H_是自定义的一个宏，该宏的名称并不一定要是头文件名称的大写。
只要确保每一个头文件都能定义出各自不同的宏就行了，用头文件名称的大写做宏是一个普通好习惯。
但是#pragma once的效率更高。不过并非所有的编译器都支持pragma，所以有的文件中混用了两种声明方式。
